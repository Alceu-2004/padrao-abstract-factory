# ☕ Cafeteria - Abstract Factory

Projeto simples em Java demonstrando o padrão de projeto **Abstract Factory**.

## 📌 Objetivo
Criar famílias de objetos relacionados (bebidas quentes e frias) sem depender das classes concretas.

## 🧩 Estrutura
- **model**: interfaces e implementações (Cafe, Cha)
- **factory**: fábricas (abstrata e concretas)
- **test**: testes unitários

## 🧪 Testes
Para executar os testes no IntelliJ:

- Clique com o botão direito na classe `FactoryTest`
- Selecione **Run 'FactoryTest'**

## 🚀 Execução
Rodar a classe:

Main.java
